What This Extension Does

Mindful Instagram is a Chrome extension that blocks distracting features on Instagram (Reels, Explore, feed, Stories) while keeping Messages, Create, and Profile accessible. It helps you use Instagram intentionally for communication, not mindless scrolling.

Data We Collect

None. Mindful Instagram does not collect, transmit, store, or share any personal data whatsoever. We have no servers, no analytics, no telemetry, and no way to access anything about you or your browsing.

Data Stored Locally on Your Device

The extension stores the following data in your browser's local storage (chrome.storage). This data never leaves your device:

Settings and preferences — toggle states (Home Feed, Stories, Notifications), Focus Lock status, cooldown duration, daily use counts.
Usage tracking data — time spent on Instagram per day and number of times Instagram was opened. Stored as anonymous counters (e.g., "usage_2026_02_15: 840"), not tied to any identity.
Housekeeping keys — last daily reset date, toggle reminder flags. These are automatically cleaned up.

All locally stored data is automatically purged after 30 days. You can also clear it at any time by removing the extension or clearing your browser data.

Instagram API Usage

The extension makes one request to Instagram's direct messaging API endpoint (/api/v1/direct_v2/inbox/) to display the names of your recent message threads on the overlay screen. This request uses your existing Instagram session cookies and goes directly from your browser to Instagram — it does not pass through any external server. The response is cached locally for 60 seconds, then discarded.

Permissions Explained

storage — saves your preferences and usage counters locally in the browser.
alarms — runs periodic checks (once per minute) for Focus Lock expiry, daily resets, and toggle auto-off timers.
host_permissions (instagram.com) — allows the content script to run on Instagram pages to hide distracting elements and show the overlay.

No other permissions are requested. The extension cannot access your browsing history, other websites, or any data outside of instagram.com.

Third-Party Services

None. The extension does not use any third-party analytics, advertising, crash reporting, or data processing services. Google Fonts is loaded for the popup interface styling — this is a static CSS resource and does not involve any user data transmission.

Data Sharing and Selling

We do not sell, transfer, or share any user data with any third party for any reason. There is no data to share — we collect nothing.

Children's Privacy

The extension does not knowingly collect any information from anyone, including children under 13. Since no personal data is collected at all, COPPA compliance is inherent.
